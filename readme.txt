Project Task Manager

Cara Penggunaan:

Lakukan login menggunakan akun terdaftar.
Apabila belum memiliki akun, lakukan register menggunakan alamat email dan password
Setelah berhasil register lakukan login menggunakan akun tersebut

Setelah berhasil login pilih tab List
Nanti akan ditampilkan task yang terdaftar.
Tambahkan task baru dengan menggunakan tombol Add New Task
Masukan Name, Priority, StartTime, EndTime, dan Active
Klik Create nanti task baru akan ditambahkan
Klik nama kolom untuk melakukan sort sesuai kolom tersebut
Gunakan fitur search untuk mencari task sesuai nama task

Pada list task terdaftar terdapat tombol di kanan
Tombol Edit untuk mengubah data dari task terdaftar dan tombol Done untuk menyelesaikan task yang sedang aktif
Untuk melakukan edit klik tombol Edit nanti akan ditampilkan form berisi task yang dipilih
Ubah parameter yang diinginkan. Setelah selesai klik Edit untuk menyimpan perubahannya.

Apabila tanggal sudah mendekati batas akhir task maka akan ada peringatan bahwa sudah mendekati deadline
Apabila tanggal sudah melewati batas akhir task maka akan ada peringatan bahwa task sudah terlambat
Jika task sudah selesai dapat menekan tombol Done untuk mengubah status task menjadi tidak aktif

Cara Testing:

Login:

Akses menu tanpa melakukan login.
Akses menu setelah melakukan login.

Klik nama kolom untuk sorting, klik lagi untuk melakukan sort berulang2. Lakukan untuk setiap kolom.
Lakukan search untuk nama task yang terdaftar.
Lakukan search untuk nama task yang tidak terdaftar.

Saat menambah task baru kosongkan form.
Tambah task baru dengan mengisi form.

Tambahkan task dengan EndTime yang berbeda satu hari setelah tanggal sekarang.
Tambahkan task dengan EndTime yang berbeda satu hari sebelum tanggal sekarang.

Untuk task aktif klik tombol Done.
Untuk task tidak aktif edit ganti jadi aktif.

Tambahkan task sampai lebih dari dari 5 (sesuai pagesize di config)
